# 上号 v1.0.1 多人连接热修复

`v1.0.1` 修复了正式版中第二位好友进入频道时可能触发的渲染器崩溃，并补齐信令会话隔离与屏幕分享细节。建议所有 `v1.0.0` 用户升级。

## 多人连接与语音

- 修复远端媒体流更新时重复绑定同一个音频元素造成的 Chromium `InvalidStateError`。第二位及后续成员加入时不再因此崩溃或卡在重连。
- 远端声音改为直接从 `MediaStream` 建立音频处理节点，保留成员独立音量、闭耳和输出设备能力。
- 每个房间客户端使用独立信令会话 ID，旧连接的关闭、错误和延迟消息不会再干扰新连接。
- 合并同一时刻的重复进入请求，避免一次点击被多个界面事件重复执行。

## 界面修复

- 扬声器关闭图标改为完整清晰的静音标记，不再出现残缺声波。
- 本机屏幕分享现在也可以打开独立观看窗口；观看窗口启用内容保护，避免采集自身形成无限镜像。

## 验证

- 通过共享类型、信令、WebRTC 与桌面端 TypeScript 检查。
- 通过 89 条桌面回归测试，包括双成员房间同步、远端音频节点和新旧信令会话竞态。
- 通过生产构建与 Windows 安装包运行时检查。

## 安装

下载 `ShangHao-1.0.1-Setup-x64.exe` 后直接覆盖安装。软件内检测到更新后，也可以自动退出并安装新版。
