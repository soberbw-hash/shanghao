# 上号 v0.1.7

这版是把上一轮已经改好的启动保护、连接模式、试音、提示音、诊断和 Windows 图标链路正式打成可下载安装的版本。

## 这次更新

- 修了部分机器点击图标后没有窗口、后台却有进程的问题
- 启动阶段补了 fallback 页面、安全模式和更完整的日志提示
- 首页加入三种手动连接模式：
  - 房主直连
  - Tailscale
  - 云中继
- 开房 / 加入的按钮反馈更明确，失败时会直接说人话
- 加了首页试音，可以直接听自己麦克风回放
- 提示音补齐了：
  - 开麦
  - 关麦
  - 成员进入
  - 成员退出
  - 连接成功 / 失败
- 设置页补了网络诊断、日志目录、诊断包导出、提示音开关
- 修了 Windows 图标链路，重新生成了多尺寸 `icon.ico`

## 下载后怎么用

1. 打开首页先选连接模式
2. 房主点“开启房间”
3. 把房间地址复制发给朋友
4. 朋友粘贴后点“立即加入”

## 适合谁更新

- 点开软件偶尔白屏的人
- 打开后没有窗口、只有后台进程的人
- 开房和加房时总觉得“没反应”的人
- 经常开 Clash / Meta / Mihomo / TUN 的人
