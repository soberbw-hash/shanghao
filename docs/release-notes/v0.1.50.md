# 上号 v0.1.50

这是一次完整的稳定性、安全、语音质量与视觉系统收口，不是中间过渡版。

## 连接与安全

- 固定频道服务端绑定真实 Socket 身份，阻止冒充发言、改状态、离开和转发音频。
- 增加访问 Token、消息大小限制、速率限制、人数上限、协议校验和异常连接隔离。
- 聊天保留最近 100 条，使用原子写入、备份恢复和服务重启恢复。
- 增加 WSS、Caddy、systemd 与 Ubuntu 一键部署方案。
- 诊断页新增 Relay 延迟、TURN、WebRTC、语音路径、丢包、抖动和音频兜底状态。

## 语音与日常使用

- 本地加入 RNNoise 智能降噪，性能过载时自动回退浏览器基础降噪，不会中断通话。
- 增加四阶 90/120 Hz 低频风噪抑制、五段语音塑形、32 kHz 默认采样率和 5 秒本地麦克风体检。
- 聋音与麦克风状态改为原子联动；按键说话不会绕过静音或聋音状态。
- 每位好友支持 0%–200% 独立音量、单独静音和真实音频节点控制。
- 只有系统连续空闲 30 分钟才会自动进入“离开一下”并闭麦，手动离开不会被自动拉回。
- 增强设备拔插恢复、全局按键说话、系统通知、录音标记和窗口位置记忆。

## 界面与交互

- 首页收敛为固定服务器地址、昵称和“上号”主操作，删除旧连接模式、AI、便签、随机昵称和一句话状态残余。
- 房间、聊天、悬浮窗、设置与屏幕分享统一为清晰克制的浅色玻璃系统，支持减少动态、减少透明、增强对比和 100%/110%/125% UI。
- 昵称与个人延迟分行显示；延迟颜色按 80/150/250 ms 分级。
- 聊天支持日期分隔、连续消息合并、新消息提示和稳定的 100 条历史。
- 44 px 胶囊悬浮窗使用角色脸部裁切，支持相同动物，不再伪造空房头像。
- 动画统一为快速、可中断、无 `transition: all`、无 `scale(0)` 的非线性节奏。

## 工程与发布

- 增加 ESLint、Prettier、React Hooks、Promise 与 Electron/Node 安全规则。
- 增加主分支 CI、CodeQL、Dependabot、Release 元数据校验、安装包内容校验和 SHA256 校验文件。
- Windows Release 只保留安装器、自动更新元数据、blockmap 与校验文件。
