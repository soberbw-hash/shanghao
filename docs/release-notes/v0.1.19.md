# 上号 v0.1.19

这版是连接配置修复版。

## 修复

- 云中继地址现在会自动纠正常见输入：
  - `1.2.3.4:43821` 自动变成 `ws://1.2.3.4:43821/`
  - `http://1.2.3.4:43821/health` 自动变成 `ws://1.2.3.4:43821/`
  - `https://relay.example.com` 自动变成 `wss://relay.example.com/`
- 继续保留地址候选重试、WebRTC STUN、信令音频兜底。

## 连接建议

如果房主直连和 Tailscale 都连不上，优先用腾讯云中继。双方都主动连接云服务器，不需要房主电脑开放公网端口。
