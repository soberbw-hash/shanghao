# 上号 v0.1.9

这版主要是把安装版启动和图标链路收干净。

## 已修复

- 修复安装版启动时主进程报错的问题
- 修复 `@achingbrain/nat-port-mapper` 的模块加载方式，避免打包后直接崩溃
- 修复主进程、托盘、启动兜底页里的乱码提示
- 统一 Windows 安装包、可执行文件、桌面快捷方式的图标链路
- 安装包可执行文件名统一为 `ShangHao.exe`
- 补齐运行时需要的托盘图标资源打包

## 下载

- Windows 安装包：`ShangHao-Setup.exe`

## 安装建议

- 如果你机器上已经装过旧版，建议先退出旧的“上号”进程，再安装这一版
- 如果桌面上还残留旧快捷方式，安装新版后删除旧快捷方式，再从开始菜单重新发送到桌面即可
