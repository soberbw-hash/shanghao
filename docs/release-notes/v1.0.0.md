# 上号 v1.0.0 正式版

`v1.0.0` 是上号面向固定好友小圈子的第一个正式版本。它把过去多个测试版本中的连接、语音、屏幕分享、交互和安装更新能力统一收口为一套可长期使用的固定开黑频道体验。

## 连接与语音

- 客户端只保留固定服务器连接，移除房主直连、Tailscale 和旧连接模式残余，第一次填好地址后即可一步上号。
- 强化信令断线恢复、成员快照自愈、媒体重新协商、音频背压和低带宽语音兜底，三人同时在线时保持完整音频路由。
- WebRTC 根据丢包和抖动自动在正常、受限、严重弱网三档切换，动态调整 Opus 音频码率、屏幕分享帧率与分辨率。
- Opus 默认启用带内纠错、静音省流和 32 kHz 语音上限；RNNoise、四阶低切与语音均衡共同降低键盘声、风噪和持续底噪。
- 麦克风试音改为实时监听，并可通过智能校准自动给出适合当前房间环境的输入阈值。

## 屏幕分享与日常使用

- 屏幕分享支持 720p/1080p、系统音频、多人分享切换和独立可拖动缩放的观看窗口。
- 本机分享使用安全占位预览，避免分享上号窗口时产生无限镜像；远端画面可自然过渡到独立窗口。
- 支持按键说话、设备拔插恢复、成员独立音量、原生系统通知、录音与可读的精彩时刻标记。
- 只有系统真实连续空闲 30 分钟才会自动进入“离开一下”，并完整保留玩家原来的静音状态，不会误开麦。

## 设计与动画

- 首页、房间、聊天、设置、悬浮窗和屏幕分享统一为清晰克制的浅色分层材质。
- 建立统一慢速非线性 Motion OS：页面、面板、按钮、消息、状态和角色动作使用一致的空间曲线与弹簧节奏。
- 麦克风、扬声器、录音、分享、邀请和设置图标拥有与含义匹配的动态反馈，不再只是统一缩放。
- 五个工位保持稳定座位仲裁；角色拥有说话光晕、待机观察、伸展、喝水、敲一敲反应和自然进离场反馈。
- 聊天统一左对齐头像与昵称，加入日期分隔、连续消息合并和更自然的发送过渡。

## 稳定性与发布

- 清理旧房间组件、旧连接模型、AI 残余和无用设置，减少维护路径和安装包冗余。
- 修复窗口销毁后的主进程事件、启动白屏、最大化、托盘恢复、覆盖安装和自动更新退出链路。
- Release 经过 ESLint、Prettier、TypeScript、87 条回归测试、生产构建、三人音频验证和 Windows 安装包运行时校验。
- Windows Release 保留安装器、自动更新元数据、增量更新 blockmap 与 SHA256 校验文件，不发布无关平台资产。

## 安装

下载 `ShangHao-1.0.0-Setup-x64.exe` 后直接安装。已安装旧版本的用户可以覆盖安装；软件内发现更新后也可以自动退出、安装并重新打开。
