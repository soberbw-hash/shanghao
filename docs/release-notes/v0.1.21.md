# 上号 v0.1.21 连接链路修复版

这一版集中修复好友跨网络加入、WebRTC 建连、邀请地址误分享和发布图标链路。

## 连接修复

- 默认 ICE 改为国内 STUN 优先，并保留海外 fallback。
- 缓存早于 remoteDescription 到达的 ICE candidate，避免乱序导致建连失败。
- WebRTC 真正进入 connected 后才停止对应成员的信令音频 fallback，避免双路音频、回声和重复声音。
- Tailscale 优先分享 `100.x` 地址，明确处理未登录、已停止和没有 Tailscale IP 的状态。
- 房主直连只允许复制已验证公网地址，不再把 LAN、loopback 或未验证地址发给跨网络好友。
- 新增“临时公网”模式，可通过 Cloudflare Quick Tunnel 生成临时 `wss://` 地址。
- 云中继测试同时检查 `/health` 和 WebSocket，可自部署在 Ubuntu 公网服务器。

## 诊断与发布

- 诊断包新增房主会话、候选地址、Fake-IP/TUN、Cloudflare 和云中继状态。
- Windows EXE、安装包和快捷方式改用缓存隔离的 v3 图标资源。
- 新增 Windows/macOS GitHub Actions 自动打包发布流程。

## 注意

- Cloudflare Quick Tunnel 可能被部分代理、TUN 或网络环境阻断；遇到 TLS/WSS 不可达时，请使用自部署云中继。
- macOS 包目前未签名，首次打开请在 Finder 中右键选择“打开”。
