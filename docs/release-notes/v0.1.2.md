# 上号 v0.1.2 中文预览版

上号是一款给固定好友小圈子使用的 Windows 私人开黑语音工具。

![上号首页](https://raw.githubusercontent.com/soberbw-hash/shanghao/main/docs/assets/release-home.png)
![上号设置页](https://raw.githubusercontent.com/soberbw-hash/shanghao/main/docs/assets/release-settings.png)

## 这次更新

- 修复 Windows 安装包首次启动时报 `Cannot find module ... index.js` 的主进程入口问题
- 重新整理中文文案和发布说明，GitHub Release 页面现在会正常显示中文

- 保留现有的单房间、Tailscale 优先、纯语音、手动录音的产品方向

## 下载后怎么用

1. 所有成员先安装并登录同一个 Tailscale 网络
2. 房主打开上号后点击“开启房间”
3. 把房主地址发给好友，好友粘贴到首页输入框后点击“加入房间”

## 当前能力

- 固定 1 个房间，最多 5 人语音
- 自由麦、按键说话、静音、输入输出设备切换
- 成员说话高亮、单独音量调节、断线自动重连
- 手动开始 / 停止录音，导出 `.m4a / AAC`
- 托盘常驻、全局静音快捷键、Tailscale 检测与引导

## 使用说明

- 当前版本优先支持 Windows
- 更适合熟人小圈子，不是频道型社交语音软件
- 如果远程加入不稳定，优先检查 Tailscale 是否已连接到同一个 tailnet
