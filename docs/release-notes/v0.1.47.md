# 上号 v0.1.47

这一版把连接入口彻底收敛为固定频道服务器，并完成媒体、动效和房间互动收口。

## 连接与媒体

- 客户端只保留服务器地址，不再展示房主直连、Tailscale 或临时公网入口。
- 屏幕分享新增流畅、均衡、清晰三档，默认 720p / 15fps / 420kbps。
- Windows 屏幕分享支持系统音频，并与麦克风混合发送。
- 多人同时分享时可切换查看，不再只能看到第一路画面。
- 服务器画面兜底进一步降低分辨率、帧率和单帧大小，减少弱服务器压力。

## 语音与快捷键

- PTT 改为全局按下/松开监听，游戏在前台时也能按住说话。
- 好友上线和“敲一下”新增 Windows 原生通知。
- 录音新增 F8 精彩时刻标记，并在录音旁生成时间点 JSON。

## 场景与动效

- GSAP、Framer Motion 和 CSS 统一为一套克制的弹簧、时长和缓动参数。
- 状态灯会随连接、重连和断开切换颜色。
- 新增一句话状态、临时表情、共享便签、游戏图标和敲击工位反馈。
- 场景会根据本地时间切换日间、傍晚和夜间光线。
- 游戏运行时自动降低玻璃模糊开销；游戏中或屏幕分享中不再自动离座。
