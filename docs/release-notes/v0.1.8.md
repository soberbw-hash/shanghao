# 上号 v0.1.8

这版是一次收口更新，重点把“能不能稳定开起来、能不能把地址发出去、能不能看懂当前状态”这几件事做扎实。

## 这次更新了什么

- 修好了房主直连模式下开房链路，补上公网地址、端口映射、可达性探测和更明确的失败原因
- 把云中继模式补成了独立流程，和直连、Tailscale 不再混在一起
- 首页改成更紧凑的双列布局，顶部去掉重复品牌和重复提示
- 试音链路重构为本地监听，新增原声 / 处理后切换、输入阈值和采样率高级设置
- 设置页补上了更新检查、日志目录、诊断包导出、网络模式信息和高级音频设置
- 启动和白屏保护继续补强，失败时会给出可见提示和日志位置
- 正式图标已经统一到安装包、任务栏、Alt+Tab、托盘和应用内品牌资源

## 这一版更适合谁

- 想自己当房主，优先试试房主直连的人
- 已经长期在用 Tailscale 的固定好友小队
- 网络环境复杂、经常开 Clash / Meta / Mihomo / TUN，需要中继兜底的人

## 使用提醒

- 房主直连如果当前网络条件不足，会明确提示你改用 Tailscale 或云中继
- 如果加入失败，先去设置页导出诊断包，再把包发出来排查会更快
- 首次安装仍然需要先填昵称和头像，房间里才好认人
