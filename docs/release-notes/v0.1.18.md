# 上号 v0.1.18

这版继续收口“朋友加入失败”的排查和云中继可用性。

## 重点

- 信令服务增加 `/health` 健康检查，方便腾讯云部署后确认端口是否真的可访问。
- 补充腾讯云中继验证步骤，明确用 `curl http://服务器IP:43821/health` 判断安全组、防火墙、服务是否通。
- 保留 v0.1.17 的连接修复：
  - 地址候选自动重试
  - WebRTC 默认 STUN
  - 信令音频兜底

## 重要说明

如果朋友连不上房主直连，通常是房主电脑的 `43821` 端口无法从外网访问。没有公网 IP、没有端口映射、Windows 防火墙拦截、校园网/公司网/运营商 NAT 都会导致这个问题。

最稳方案是使用云中继：双方都主动连接腾讯云服务器，不要求房主电脑暴露端口。
