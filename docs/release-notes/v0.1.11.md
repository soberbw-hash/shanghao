# 上号 v0.1.11

![上号图标](https://raw.githubusercontent.com/soberbw-hash/shanghao/main/docs/branding/github-avatar.png)

这版是一次高优先级修复收口，重点把“房主直连卡住”“首页布局不稳”“图标主体偏小”这几件最影响可用性的地方收了。

## 这次重点更新

- 房主直连改成“两阶段启动”
  - 先启动本地房间
  - 再异步探测公网 IP、端口映射和外网可达性
- 修复 `direct_host` 一直停在“正在开启...”的问题
- 首页去掉重复品牌区，头部更紧凑
- 左侧主操作区和右侧临时聊天区改成等高双列
- “开启房间”按钮位置固定，不再随模式切换上下跳动
- 开黑位恢复成稳定一整排 5 列，不再换行露半截
- 桌面图标、快捷方式、开始菜单、任务栏图标重新导出，主体占比更大
- 说明文档和品牌资源一并整理

## direct_host 现在怎么工作

房主直连现在不再把公网探测当成唯一前置条件。

流程变成：

1. 先启动本地 signaling server 和房间
2. 房主本机立即通过本地回环地址进入房间
3. 后台继续探测：
   - 公网 IP
   - UPnP
   - NAT-PMP
   - 外网可达性
4. 如果探测通过，再生成可分享地址
5. 如果探测失败，房间也不会卡死，只会明确提示建议切到 Tailscale 或云中继

## 升级建议

如果你本机之前装过旧版，建议：

1. 先退出正在运行的旧版
2. 安装新版本
3. 如桌面仍是旧快捷方式，删掉后从开始菜单重新发送到桌面

## 出问题时怎么排查

到设置页可以直接：

- 导出日志
- 导出诊断包
- 打开日志目录

如果房主直连不可用，优先改用：

1. Tailscale
2. 云中继
