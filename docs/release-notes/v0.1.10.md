# 上号 v0.1.10

这版主要把首页、直连开房链路和图标链路一起收口了。

更新内容：

- 修复房主直连开启房间时容易卡住的问题
- 房主本机加入改走本地 loopback，不再错误回连公网分享地址
- 首页重构为更紧凑的双栏布局，去掉重复品牌区和右侧试音卡
- 主页右侧改为轻量临时聊天，只支持文字和 emoji
- 设置页移除不该交给用户决定的提示音开关和自动复制开关
- PTT 按键支持自定义、清空和恢复默认
- 默认窗口尺寸继续加大，减少首屏滚动
- 重建 Windows 图标资源，补齐多尺寸 `icon.ico`
- 统一 `icon.svg` / `logo-ui.svg` / `tray` 图标资源
- 新安装包版本：`ShangHao-Setup.exe`

建议：

- 如果你之前装过旧版，先退出旧进程，再安装这版
- 如果桌面快捷方式图标还是旧缓存，删掉旧快捷方式后从开始菜单重新发送到桌面
