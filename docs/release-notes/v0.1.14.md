## 上号 v0.1.14

这版是一次 P0 热修，重点只收两件事：连通性和图标链路。

### 这次修了什么

- 修复 `房主直连` 在房间本地已启动后，仍然因为公网自检未完成而拿不到可分享地址的问题。
- `direct_host` 现在会区分：
  - 房间本地已启动
  - 正在检测公网直连能力
  - 已生成候选地址
  - 公网直连已验证可用
  - 当前网络拿不到可分享地址
- 房间页继续保持右侧为临时聊天区，连接方式和房间地址留在左侧。
- 更新 Windows 图标链路，安装器、EXE、窗口图标和桌面快捷方式统一使用新的正式图标资源。
- 桌面快捷方式现在使用独立的 `shanghao-shortcut.ico`，减少 Windows 复用旧缓存的概率。

### 建议这样复测

1. 房主和好友都升级到这个版本。
2. 房主点“开启房间”后，先确认页面会明确显示“房间已启动”。
3. 如果是 `房主直连`，继续观察地址区是否进入：
   - 正在检测
   - 已生成候选地址
   - 或明确提示改用 Tailscale / 云中继
4. 如果桌面还是旧图标，请卸载旧版后重新安装，并从开始菜单重新发送一个快捷方式到桌面。
