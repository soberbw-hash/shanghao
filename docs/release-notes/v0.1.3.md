# 上号 v0.1.3 中文预览版

上号是一款给固定好友小圈子使用的私人开黑语音工具。

![上号首页](https://raw.githubusercontent.com/soberbw-hash/shanghao/main/docs/assets/release-home.png)
![上号设置页](https://raw.githubusercontent.com/soberbw-hash/shanghao/main/docs/assets/release-settings.png)

## 这次更新

- 修复“点击开启房间没反应”的整条链路
- 房间真正连上之后才跳转到房间页
- 开房 / 加房增加加载态、禁用态、错误 Toast 和日志
- 首次启动改为先完成昵称和头像初始化
- 头像支持本地选择、预览、保存和重置
- 首页改成更轻的白色极简布局
- 成员区改成紧凑的“开黑位”
- 设置页补齐资料、音频、快捷键、网络和诊断项
- 内置 HarmonyOS Sans，本地打包即可使用

## 下载后怎么用

1. 所有成员先安装并登录同一个 Tailscale 网络
2. 房主打开上号，完成资料初始化后点击“开启房间”
3. 把邀请地址发给朋友
4. 朋友把地址粘贴到首页右侧，点击“加入房间”

## 当前版本能力

- 固定 1 个房间，最多 5 人语音
- 自由麦、按键说话、静音
- 输入 / 输出设备切换
- 说话状态高亮、单独音量调节
- 断线自动重连
- 手动录音并导出 `.m4a / AAC`
- 托盘常驻、全局静音快捷键、Tailscale 检测

## 使用提醒

- 当前版本优先支持 Windows
- 更适合熟人小圈子，不是频道型社交语音软件
- 如果远程加入不稳定，优先检查 Tailscale 是否已经连到同一个 tailnet
