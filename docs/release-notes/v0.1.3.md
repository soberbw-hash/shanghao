# 上号 v0.1.3 中文预览版 🎮

一句话先说清：
朋友喊你“上号”，你打开软件，直接进语音。少折腾，少废话，开黑更快。🎧

上号是一款给固定好友小圈子用的私人开黑语音工具。

## 这次更新了什么 ✨

- ✅ 修好“点击开启房间没反应”这条链路
- ✅ 真正连上房间后才跳转，不再假进房
- ✅ 开房 / 加房补上加载态、禁用态、错误提示和日志
- ✅ 第一次启动改成先填昵称和头像，更容易认人
- ✅ 头像支持本地选择、预览、保存和重置
- ✅ 首页改成更清爽的白色极简风
- ✅ 成员区改成更像开黑位的紧凑布局
- ✅ 设置页补齐资料、音频、快捷键、网络和诊断项
- ✅ 内置 HarmonyOS Sans，打包后直接能用

## 这软件适合谁 👀

- 👬 固定几个朋友一起打游戏
- 🎮 平时不常驻，开黑时再打开
- 🧼 不想用太重、太吵、太像社交平台的语音工具
- 🌐 愿意优先走 Tailscale 私网环境

## 下载后怎么用 🚀

1. 🛰️ 所有成员先安装并登录同一个 Tailscale 网络
2. 👤 第一次打开上号，先填昵称和头像
3. 🏠 房主点击“开启房间”
4. 🔗 把邀请地址发给朋友
5. 🚪 朋友把地址粘贴到首页右侧，点击“加入房间”

## 现在能做什么 🎛️

- 👥 固定 `1` 个房间，最多 `5` 人语音
- 🎙️ 自由麦、按键说话、静音
- 🎚️ 输入 / 输出设备切换
- 💡 说话状态高亮、单独音量调节
- 🔁 断线自动重连
- ⏺️ 手动录音并导出 `.m4a / AAC`
- 📌 托盘常驻、全局静音快捷键、Tailscale 检测

## 使用提醒 📎

- 🖥️ 当前版本优先支持 Windows
- 🧑‍🤝‍🧑 更适合熟人小圈子，不是频道型社交语音软件
- 🔍 如果远程加入不稳定，先看看 Tailscale 是不是已经连到同一个 tailnet
