# 上号 v0.1.48

这是一版面向真实多人使用的连接稳定性与体验收口版本。

## 连接

- 信令断线不再在第 6 次后永久放弃，改为带随机抖动的持续退避重连。
- WebRTC 连接失败、断开或超时后会自动重建点对点会话并重新协商。
- 服务端可下发短期 TURN HMAC 凭据，客户端会同时保留公共 STUN 与 TURN。
- WebSocket 语音兜底改用 μ-law 编码，带宽约为旧 PCM16 方案的一半。
- 慢客户端只丢弃过期的实时音视频帧，不再拖累整个固定频道。
- 增加 TURN、媒体重建、实时丢帧与每位好友音频路径诊断。

## 体验

- 已完成资料与服务器配置的用户改为一步“上号”，不再每次重复填写。
- 顶部显示实时服务器延迟，备用语音通道有明确状态反馈。
- 液态玻璃集中用于导航与控制层；场景和聊天改为更清晰、性能更稳的内容材质。
- 收紧首屏留白、网格与面板阴影，保留白色极简风并强化空间层级。

## 部署

- 新增 `scripts/install-turn.sh`，可在 Ubuntu / 腾讯云上一键安装 coturn。
- 新增 TURN 安全组、环境变量、健康检查与排障文档。
- 清理未实现的“频道码”说明，客户端仍只需填写一个固定服务器地址。

协议版本升级为 `4`，客户端和服务器需要同时更新。
