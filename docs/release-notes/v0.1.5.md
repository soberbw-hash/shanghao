# 上号 v0.1.5

这版重点把“有人打开白屏、有人开房没反应、加入失败看不懂原因”这三件事一起补齐了。

## 本次更新

- 修复启动阶段白屏，加入启动页、安全模式、错误恢复页和更完整的启动日志
- 修复“开启房间”反馈弱的问题，补齐加载态、失败恢复、成功提示和详细日志
- 房主地址改成固定端口优先，优先使用 MagicDNS，其次回退 Tailscale IP / 局域网 IP
- 新增强化版加入诊断，加入失败时会更容易判断是地址问题、Tailscale 问题还是端口不通
- 首次安装现在必须先填写昵称和头像，房间成员优先显示真实头像
- 首页、房间页、设置页继续统一成白色极简风，成员位更紧凑
- 修正仓库首页和发布说明里的中文乱码

## 下载安装

- Windows 安装包：`ShangHao-Setup.exe`
- 仓库地址：[shanghao](https://github.com/soberbw-hash/shanghao)

## 升级提醒

- 如果你装过旧版，建议直接覆盖安装到最新版
- 如果加入失败，先看顶部状态和提示，再检查 Tailscale 是否连接到同一个 tailnet
- 如果启动进入安全模式，也可以先正常使用，再到设置里导出日志
