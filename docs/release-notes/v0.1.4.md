# 上号 v0.1.4

这版重点只做一件事：把“点击开启房间后白屏”的问题修掉，让大家能更稳地直接进语音。

本次更新

- 修复开启房间后房间页白屏的问题
- 稳定成员状态同步，减少异常刷新
- 补强房间页渲染兜底和日志记录
- 优化房间启动后的连接链路稳定性

下载安装

- Windows 安装包：`ShangHao-Setup.exe`
- 仓库地址：[shanghao](https://github.com/soberbw-hash/shanghao)

使用提醒

- 房主点击“开启房间”后，会正常进入房间页
- 朋友仍然通过房主分享的地址加入
- 如果你本机已经装过旧版，建议直接覆盖安装到最新版
