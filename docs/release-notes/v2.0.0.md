# 上号 2.0.0

这是“上号”第一版完整动画大版本，同时重点修复多人语音、屏幕分享和重连体验。

## 这一版有什么

- 角色从场景左侧进入，使用八帧交替步态跑向工位。
- 切换座位采用可打断的最短避障路径，不再坐着椅子漂移。
- 统一弹窗、消息、按钮、状态和场景反馈的非线性动效。
- 冰川蓝环境光、可读玻璃边缘和更克制的层级材质。
- 第四、第五位好友加入时校验双向收音，修复后加入者单向听不见。
- 远端成员共用音频混音器，降低多人房间音频上下文和设备切换压力。
- WebRTC 异常时按成员启用低带宽语音兜底，并在恢复后自动停止。
- 屏幕分享使用直接媒体流，支持 720p / 1080p、系统音频和独立观看窗口。
- RNNoise、低切、语音均衡、32 kHz、FEC/DTX 和弱网自适应继续生效。
- 重连遮罩延迟出现，Toast 自动去重并限制数量，不再连续遮挡操作。

## 安装

下载 `ShangHao-2.0.0-Setup-x64.exe`。安装程序支持覆盖安装；若旧版本仍在托盘后台，
安装器会先请求旧进程退出再继续。

## 说明

上号 2.0 仍然只面向 Windows 固定好友小圈子。所有人首次填写同一个固定服务器地址，
以后打开软件点击“上号”即可进入。
