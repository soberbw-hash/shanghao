# 上号 v0.1.13

这版是一次连接可用性和房间页布局收口更新，重点把“开房后聊天区消失、地址占错位置、快捷方式还是老图标”这几件最影响体验的地方修掉了。

## 这次重点修复

- 房间页右侧恢复为固定临时聊天区，开房前后都不会消失
- 房间地址、连接方式、房主状态摘要全部移回左侧区域
- `direct_host` / 本地开房状态表达继续收口
  - 房间已启动
  - 地址探测中
  - 地址已可分享
  - 当前网络不支持公网直连
- 桌面快捷方式、开始菜单快捷方式改为显式使用新的独立图标资源
- 新增快捷方式专用 `shortcut.ico`，减少 Windows 继续复用旧图标缓存的问题

## 房间页现在怎么排

- 左侧：房间状态、连接方式、分享地址
- 右侧：临时聊天
- 下方：开黑位、录音状态、底部控制条

这样开房之后，聊天不会再被地址卡挤掉，地址也不会再跑去占右侧整块区域。

## 图标这次怎么处理

这次不再只依赖 EXE 内部图标。

安装器会显式给桌面和开始菜单快捷方式绑定新的 `shortcut.ico`，并主动刷新一次 shell 图标通知，尽量把旧缓存的影响压到最低。
