# 上号 v0.1.49

这是一版专门修复 Windows 覆盖安装与应用内自动更新的版本。

## 覆盖安装

- 修复安装器在应用没有运行时，反而启动旧版 `ShangHao.exe` 的问题。
- 只有检测到旧进程时才发送退出请求，旧版不支持退出参数也不会阻塞安装器。
- 安装前会依次完成优雅退出、短暂等待和按安装目录清理，不再要求反复点击“重试”。
- 清理进程时排除安装器自身，避免卸载与覆盖流程互相误杀。

## 自动更新

- 应用内点击更新后改为静默安装，不再弹出普通安装向导。
- 更新前主动关闭托盘、悬浮窗、游戏检测和全局快捷键。
- 安装完成后自动重新启动上号。
- 增加 4 秒强制退出保险，防止原生钩子或隐藏窗口拖住安装器。

## 验证

- 覆盖现有 `0.1.47` 安装进行真实升级验证。
- 验证应用未运行、应用正在后台运行两种覆盖场景。
- 保留 `latest.yml` 与 blockmap，继续支持后续应用内差分更新。
